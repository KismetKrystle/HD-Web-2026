# H.E.R. DAO Website

Official web presence of H.E.R. DAO.

Live Site: https://her-dao.xyz

## About

H.E.R. DAO is committed to increasing diversity and inclusion in Web3, creating a sustainable pipeline for diverse developers and fostering innovation through education and community.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- GSAP
- CSS Modules
- Deployed via Vercel

## Architecture

Structured using the Next.js App Router pattern with modular sections and reusable layout components.

## Development

Maintained by the H.E.R. DAO core team.
